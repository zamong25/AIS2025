## 📌 문서 관리 원칙

1. **체계적 분류**: 문서의 성격에 따라 적절한 폴더에 배치
2. **명확한 이름**: 문서의 내용을 명확히 나타내는 파일명 사용
3. **버전 관리**: 중요한 변경사항은 git commit으로 추적
4. **언어 표시**: 한글 문서는 파일명에 명시
